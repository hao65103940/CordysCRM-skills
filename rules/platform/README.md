# 平台级规则

> Cordys CRM 平台通用规则，任何公司都适用

---

## 说明

本目录存放 **Cordys CRM 平台的通用规则**，包括：

- 标准中国省份划分
- 通用查询逻辑
- 平台级最佳实践

这些规则对任何使用 Cordys CRM 的公司都适用，**不需要修改**。

---

## 文件列表

目前为空，通用规则已整合到 `references/api.md` 和 `references/fields.md` 中。
