# 公司级规则

> 本目录存放 **公司自定义规则**，其他公司使用时请替换成自己的规则

---

## 说明

本目录的规则是 **飞致云（Fit2Cloud）自己的业务规则**，包括：

- 区域映射（北/东/南区 + 国外）
- 常见查询场景
- 专业术语映射（产品简称/行业别名）

**其他公司使用时，请替换成自己的规则！**

---

## 替换指南

### 1. 区域映射 (`region.md`)

如果你们公司有自己的区域划分（如华北/华东/华南），修改此文件。

**示例：**
```markdown
# 我的公司区域映射

| 省份 | 区域 |
|------|------|
| 北京 | 华北一区 |
| 上海 | 华东一区 |
| 广东 | 华南一区 |
```

### 2. 查询场景 (`query-scenarios.md`)

如果你们有特殊的查询逻辑（如关联查询规则），修改此文件。

### 3. 术语映射 (`glossary.md`)

如果你们有自己的产品简称、行业别名，修改此文件。

**示例：**
```markdown
# 我的公司术语映射

| 用户用语 | 标准化术语 |
|---------|-----------|
| JS | JumpServer 企业版 |
| MK | MaxKB 专业版 |
```

## 最小化配置

如果不想用公司级规则，可以：

1. 删除本目录所有文件
2. 在 `SKILL.md` 中说明不使用公司级规则

此时所有查询都使用通用规则（`references/` 下的文档）。
