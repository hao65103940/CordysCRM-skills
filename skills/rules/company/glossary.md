## 常用 jq 命令

```bash
# 提取 ID 列表
jq -r '.data.list[].id'

# 转 JSON 数组
jq -R . | jq -s .

# 筛选金额大于 50 万
jq '[.data.list[] | select(.amount != null and .amount > 500000)]'
```
